## 技术决策

| 决策 | 选择 | 理由 |
|---|---|---|
| 框架 | Taro 3.6.38 | React 语法 → 微信小程序 |
| 状态管理 | Zustand | 轻量、TypeScript 友好 |
| 样式 | 内联 CSS (WXSS) | Taro 默认支持 |
| API 客户端 | Taro.request | 内置支持，无需额外依赖 |
| 图标 | Emoji | 轻量、跨平台一致 |
